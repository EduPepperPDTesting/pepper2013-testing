Sine (abbr. sin)

The sine of an angle in a right triangle is defined as the ratio of the length of the side opposite the angle to that of the hypotenuse. Sine is usually abbreviated as sin.
As shown by the properties of similar triangles, the sine of an angle depends only upon the measure of the angle, not upon the size of the right triangles. The value of the ratio changes as the angle changes. For this reason, this ratio is called a function of the angle A. This ratio has a fixed value for any specific angle. For example, the ratio of the length of the side opposite the 30o angle to that of the hypotenuse in a right triangle is always 0.5.

What benefits do we gain by introducing this new concept of sine function? Since the ratio is a fixed value for any specific angle, we can construct a table showing the values of the ratios for the values of a range of angles such as 0° through 90°. Then this trigonometric table can be used to facilitate computation. A measuring tape or similar tool can now be used to measure one side and a transit can be used to measure one of the acute angles in a right triangle. With the help of the trigonometric table, the other unknown sides can be determined.
Without introducing the sine function, we can still calculate the unknowns using similar triangles. However, we have to construct similar triangles every time. This is both a time-consuming and error-prone process, resulting in less accurate calculations.
